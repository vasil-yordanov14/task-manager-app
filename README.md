Task Management App

# Description

This is an simple application to organize yor tasks. (Adding new, editing and delete complete ones)
In order to create the app i used React.js.

# Installation

1.  Install dependencies: npm install
2.  Start the application: npm start

# Usage

Ð¢o use the app you need to enter a title and description for each given task. You can add an unlimited number of tasks, as well to delete those that have been completed. Also there is option to edit unfinished tasks.

# Features

Added option to download unfinished tasks. You can download them from the Download button in .txt format.
